Library - a reusable piece of code. Asset - a piece of data used by code. Here is the list of assets and libraries used by this blog. Additionally the source code of this blog is available at github.

Backend

Nim is the language used to build this site. This blog is a plain html, Nim serves a role of a static site generator. Besides the standard Nim library the following libraries are used:
  • regex - used instead of standard re library to remove external dependency on PCRE C library
  • rainbow - colored output in console
  • markdown - transform markdown files to html

Fonts

  • Aron Grotesque - headers
  • Carlito - text content
  • JetBrains Mono - code

CSS framework

Pure.css - lightweight simple framework with a touch of design and responsiveness

Code highlighting

highlight.js - pretty much all-included solution for code highlighting with a ton of supported languages and different themes

Hosting

GitHub pages - simple free solution for hosting static content like this blog
